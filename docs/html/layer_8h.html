◆ DEFAULT_BATCH_SIZE

#define DEFAULT_BATCH_SIZE   32

The default batch size if no batch size is specified.

This macro is also used to pre-allocate space when a layer is first created. In this way all required matrices/vectors are initialized and may only need reshaping later when starting training.
